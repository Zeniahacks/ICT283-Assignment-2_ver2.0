Detailed Description

Central storage and retrieval system for weather data.

Author
Elson Ang
Date
2026

This class encapsulates the STL map and the custom BST.

  • The std::map handles fast retrieval of data by YearMonth.
  • The Bst maintains a sorted record of available years, utilizing function pointers. It strictly stores pointers to dynamically allocated WeatherRecords and manages their memory.
