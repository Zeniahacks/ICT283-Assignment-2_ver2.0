Detailed Description

template<class T>
class Bst< T >

A generic Binary Search Tree implementation.

Author
Elson Ang
Date
2026

This template class implements a standard Binary Search Tree. It strictly utilizes raw function pointers for traversals to maintain compatibility with C++98/C++11 constraints and ensures the client receives read-only access (const T&) to preserve tree invariants.
